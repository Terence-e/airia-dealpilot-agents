# DealPilot — AI Sales Agents with Auto Compliance Monitoring

DealPilot is a multi-agent AI system that automates sales intelligence,
proposal generation, and compliance monitoring.

Built for the AIRIA AI Agent Challenge – Track 2.

Features
- Research Agent → gathers company intelligence
- CRM Agent → analyzes deal history
- Compliance Agent → detects regulatory risks
- Proposal Agent → generates personalized proposals
- Email Agent → drafts follow-up emails
- Dashboard → visualizes pipeline and agent activity

Tech Stack
- Airia AI Agents
- Node.js / Express
- React Dashboard
- Slack integration
- OpenAI API
